# SQL Retail Store Analysis

This project analyzes retail store sales data using Python and SQLite.

## Tools Used
- Python
- SQLite
- SQL Queries

## Files
sql_project1.py – Python script that creates the database and runs queries  
retail_store.db – SQLite database containing sales data

## Objective
To analyze retail sales and customer information using SQL queries.

## Skills Demonstrated
- Database creation
- SQL querying
- Python + SQLite integration
